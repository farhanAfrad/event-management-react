###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>Here two things happened. 1st, declaration of greeting varibale is done by doing, let greeting. 2nd, initialization is done by doing greeting = {}. {} it is a syntax of empty object, means an empty object is assigned to the variable greeting. That is why printing greeting will give an empty object.</i>

</p>
