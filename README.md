
# EVANTOo

This is a project based on coporate event management where JavaScript libray React is used. With the help of Vite a ready template to work with react is installed.


## Features

- A functional registration and login page is in the project. Where first user have to create an account by completing registration then user can login with valid credential.
- An optional login is also added where user can log in via their google account.
- Based on logged user there is two private page. In one page user can see their selected services after selection. In other page user can see their profile.
- In home page there are read more button which will redirect user to the details page. To explore read more button and details page user have to be logged in. Otherwise they can not access that page.
- There is also logout option by cliking logout button users can logged out themselves.
